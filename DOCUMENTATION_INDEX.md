## üéØ Key Standards Summary

### Container & Padding
- Use `-mx-4 md:-mx-6 lg:-mx-8` for outer containers
- Use `px-4 md:px-6 lg:px-8` for content sections
- No container class names (use consistent pattern)

### Page Headers
- Always use `<app-page-header>` component
- Support translation with `[translateTitle]` and `[translateSubtitle]`

### Grid Patterns
- Small cards: 3 columns (`lg:grid-cols-3`)
- Medium cards: 4 columns (`lg:grid-cols-4`)
- Standard gap: `gap-6`

### Mobile Optimization
- Responsive text: `text-lg md:text-xl`
- Responsive padding: `p-4 md:p-6`
- Touch feedback: `active:scale-95`

### Accessibility
- ARIA labels on interactive elements
- Keyboard navigation support
- Semantic HTML roles
