Theme Switcher

Component for switching between light/dark modes and theme colors

Features

  • Switch between Light, Dark, and Auto modes
  • Choose from 8 theme colors (Blue, Indigo, Purple, Green, Orange, Red, Teal, Pink)
  • Theme preferences are saved to localStorage
  • Auto mode follows system preference
  • Syncfusion components theme switches automatically
