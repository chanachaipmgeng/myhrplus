Chat UI Component

Chat UI component for building interactive chat interfaces. Supports message history, typing indicators, file attachments, and custom message templates (Syncfusion).

Message Handling

Handle message events with the `(messageSend)` event. You can process messages, send to server, or display in chat history.

Custom Placeholder

Customize the input placeholder text to guide users.

Responsive Design

Chat UI component is responsive by default. Use responsive height and width for mobile devices.
