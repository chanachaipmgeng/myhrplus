Overview

This guide demonstrates how to migrate legacy JSP components to modern Angular Glass Morphism components. Follow these patterns to ensure consistency across the application.

Legacy JSP Pattern

  • Solid backgrounds (bg-white, bg-gray-50)
  • Direct DOM manipulation ($('#id'))
  • Bootstrap class names (form-control, card)
  • Spinner loaders
  • Hardcoded text
  • Page refresh on navigation

Modern Angular Pattern

  • Transparent/Glass backgrounds
  • Angular directives & bindings
  • Glass components (app-glass-*)
  • Skeleton loaders
  • i18n Translation
  • SPA routing (no page refresh)

Migration Checklist

Use Standalone Components
All new components must be standalone: true
Transparent Backgrounds
Remove bg-white, bg-gray-50 from containers
Skeleton Loading
Use app-skeleton-loader instead of spinner
Reactive Forms
Use FormGroup/FormControl, not ngModel
BaseApiService
Extend BaseApiService for all services
i18n Translation
Use | translate pipe and TranslateService
