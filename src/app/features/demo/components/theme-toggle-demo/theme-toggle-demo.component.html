Theme Toggle Component

Theme toggle component for switching between light, dark, and auto modes, as well as different theme colors.

Features

Theme Modes

  • Light Mode
  • Dark Mode
  • Auto (follows system)

Theme Colors

  • Blue (default)
  • Indigo
  • Purple
  • Green
  • Orange
  • Red
  • Teal
  • Pink

Features

  • Persistent settings
  • Dropdown menus
  • Visual color preview
  • Gemini theme support
