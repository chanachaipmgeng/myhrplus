Overview

The `appStagger` directive makes it easy to add staggered animations to list items without having to write `[style.animation-delay]` repeatedly.

Best Practices

  • ✓ Use staggerDelay="0.1" for general list items
  • ✓ Use staggerDelay="0.05" for lists with many items (10+ items)
  • ✓ Use staggerDelay="0.15" for lists with few items (3-5 items) to make the effect more visible
  • ✓ This directive automatically adds the animate-fade-in-up class and sets animation-delay
