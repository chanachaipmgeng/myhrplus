Overview

`appStagger` directive ช่วยให้การเพิ่ม staggered animations ให้กับ list items เป็นเรื่องง่าย โดยไม่ต้องเขียน `[style.animation-delay]` ซ้ำๆ ทุกครั้ง

❌ Manual Way

@for (item of items; track item.id; let i = $index) {
  <div 
    class="animate-fade-in-up"
    [style.animation-delay]="(i * 0.1) + 's'">
    
  </div>
}

✅ With appStagger

@for (item of items; track item.id; let i = $index) {
  <div [appStagger]="i" [staggerDelay]="0.1">
    
  </div>
}

Best Practices

  • ใช้ staggerDelay="0.1" สำหรับ list items ทั่วไป
  • ใช้ staggerDelay="0.05" สำหรับ lists ที่มี items มาก (10+ items)
  • ใช้ staggerDelay="0.15" สำหรับ lists ที่มี items น้อย (3-5 items) เพื่อให้เห็น effect ชัดเจน
  • Directive นี้จะเพิ่ม class animate-fade-in-up และ set animation-delay อัตโนมัติ
