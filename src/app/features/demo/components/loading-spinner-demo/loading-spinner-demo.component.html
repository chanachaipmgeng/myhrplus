Advanced Features

Loading Spinner component ใช้ร่วมกับ LoadingService เพื่อควบคุมการแสดง/ซ่อน loading state

  • เพิ่ม component ใน app component หรือ layout component
  • ใช้ LoadingService.show() เพื่อแสดง loading
  • ใช้ LoadingService.hide() เพื่อซ่อน loading
  • Component จะแสดง/ซ่อนอัตโนมัติตาม state ของ service

API Reference

Loading Spinner component has no inputs or outputs. It uses LoadingService internally to manage loading state.
